# Bill Splitter

A simple Python program that calculates how much each person should pay when splitting a bill, including a tip.

## Description
This program asks the user for the total bill amount, the tip percentage, and the number of people sharing the bill. It then calculates the tip, adds it to the total bill, and shows how much each person needs to pay.

## Features
- Calculates tip based on percentage
- Adds tip to the total bill
- Splits the bill evenly among people
- Displays a clear bill summary

## Requirements
- Python 3
