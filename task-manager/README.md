# Terminal Task Manager 📝

A simple command-line task manager built with Python.

This program allows users to manage their daily tasks directly from the terminal.

## Features
- ➕ Add tasks
- 📋 View tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Tasks are saved in a file so they remain after closing the program

## How to Run

1. Clone the repository

git clone https://github.com/ShresthaSarangi/python-projects.git

2. Go to the project folder

cd python-projects

3. Run the program

python task_manager.py
